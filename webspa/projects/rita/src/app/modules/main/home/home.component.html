RITA Web es un juego para aprender a programar, en el cual podrás desarrollar un tanque de batalla robot para luchar contra otros tanques.

Utiliza el motor del popular juego Robocode, pero facilita el aprendizaje de la programación, ya que los robots se pueden crear moviendo y encastrando bloques.

Para programar tu primer robot podes ir a Mis robots


Si querés crear una batalla o ver una ya creada podés ir a Batallas


Si querés conocer más de Robocode y estrategias para crear Robots podés ir a Robo wiki


RITA Web es un producto de software libre, lo que significa que podés descargarlo, modificarlo e inclusive proponer mejoras o cambios. El código fuente se encuentra disponible aquí
